# Manual Raw Captures

This directory is reserved for intentional manual captures from PMM UI surfaces or related MakerWorld pages.

Do not auto-scrape PMM UI copy in v1.

When adding a manual capture:
- copy `template-manual-capture.md`
- rename it to describe the capture
- fill in the provenance metadata
- state whether the text is verbatim, paraphrased, or transcribed from a screenshot

These captures should be used sparingly and cited as `Manual UI capture`.
